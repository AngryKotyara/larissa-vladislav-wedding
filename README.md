# Свадебный сайт Larissa & Vladislav

Готовый сайт-приглашение с персональными ссылками, RSVP и административной панелью.

## Быстрый запуск

```bash
npm install
ADMIN_LOGIN=admin ADMIN_PASSWORD=0898kV01vladislav SESSION_SECRET=2208 PUBLIC_URL=https://larissaandvladislav.com npm start
```

Откройте:

- сайт: `http://localhost:3000`
- админ-панель: `http://localhost:3000/admin`

## Важно перед публикацией

1. Обязательно задайте `ADMIN_PASSWORD` и `SESSION_SECRET` через переменные окружения.
2. Укажите реальный адрес сайта в `PUBLIC_URL`, иначе персональные ссылки будут содержать localhost.
3. Проверьте программу дня в `public/index.html`: сейчас стоят ориентировочные 15:00, 16:00, 17:00 и 23:00.
4. При необходимости замените ссылку на карту на точный адрес ресторана.
5. Данные RSVP хранятся в `data/invites.json`. Для постоянной работы нужен сервер с сохраняемым диском и резервным копированием этого файла.

## Как пользоваться

1. Войти в `/admin`.
2. Ввести имя гостя или семьи и обращение.
3. Нажать «Создать ссылку».
4. Скопировать персональную ссылку и отправить гостю.
5. Ответ появится в таблице администратора.
6. Список можно выгрузить в CSV.

## Размещение

Проект рассчитан на Node.js 18+ и подходит для VPS, Railway, Render или любого хостинга с постоянным диском. Для serverless-хостинга файловое хранение нужно заменить на PostgreSQL, Supabase или другую внешнюю базу данных.
