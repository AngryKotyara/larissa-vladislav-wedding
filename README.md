# Свадебный сайт Larissa & Vladislav

## Состав архива

- `index.html` — сайт-приглашение.
- `admin.html` — форма администратора для создания персональных ссылок.
- `config.js` — дата, место, текст приглашения и история знакомства.
- `styles.css`, `script.js` — оформление и логика сайта.
- `admin.css`, `admin.js` — оформление и логика панели администратора.
- `assets/` — фотографии.

## Как открыть локально

Откройте `index.html` двойным щелчком. Для создания персональных приглашений откройте `admin.html`.

## Как опубликовать

Загрузите все файлы без изменения структуры на любой статический хостинг: Netlify, GitHub Pages, Cloudflare Pages, Timeweb или другой хостинг.

После публикации откройте `admin.html`, вставьте адрес опубликованного сайта в поле «Адрес опубликованного сайта» и создавайте ссылки для гостей.

## Как изменить тексты

Откройте `config.js` в любом текстовом редакторе. В нём можно изменить:

- имена;
- дату и время;
- название ресторана;
- ссылку на Яндекс Карты;
- текст приглашения;
- этапы истории знакомства.

## Персональная ссылка

Администратор вводит имя гостя, например «Анна и Сергей». Получится ссылка вида:

`index.html?guest=Анна%20и%20Сергей&greeting=Дорогие`

Имя автоматически появится на странице приглашения.

## Важно

Список созданных приглашений хранится в браузере администратора через `localStorage`. Для переноса списка используйте кнопку «Экспорт CSV».
